  • Wave

    Wave is an in-browser error labeller and contrast checker:

    Chrome: Chrome Wave extension

    Firefox (not tested):Firefox WAVE extension

  • Tota11y

    Tota11y is an in-browser accessibility checker that you can use to run an in-browser label, link and heading chcker, as well as a 'screenreader wand'

    You can find it for Chrome in the extensions library and add it via Tota11y Chrome browser plugin

    You can also install it into your site by adding this code via github
